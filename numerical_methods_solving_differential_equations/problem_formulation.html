Постановка задачи

Рассмотрим обыкновенное дифференциальное уравнение первого порядка \[y' = f(x,y)\]с начальными условиями \[y(x_0) = y_0.\]

Эту задачу называют обычно задачей Коши. Будем предполагать, что функция \(f(x,y)\) определена по первому аргументу на отрезке \([x_0,x_0 + X]\), по второму аргументу — на всей числовой прямой; в области определения функция \(f(x,y)\) непрерывна, а также выполняется условие липшицевости по второму аргументу: существует константа \(L\), такая, что для всяких \(y_1\) и \(y_2\) выполняется \[|f(x,y_1) - f(x,y_2)| \leq L|y_1 - y_2|.\] Эти условия гарантируют существование и единственность решения \(y(x)\) задачи Коши. Если, кроме того, потребовать выполнение условия продолжимости, например, выполнение условия подлинейного роста \[f(x,y) \leq \alpha(x)(1 + |y|),\] то решение определено на всем отрезке \([x_0,x_0 + X ]\).

Будем искать приближение этого решения в дискретной схеме. Разобьем отрезок \([x_0, x_0 + X]\) на \(N\) равных частей с шагом \(h =\frac{X}{N}\), обозначив узлы \(х_i = x_0 + ih, i = 0,\cdots, N\). Приближенные значения решения в узлах будем обозначать \(y_i\), в отличие от точных значений решения в узлах \(y(x_i), i = 0,\cdots, N\).

Задача состоит в конструировании алгоритмов для \(y_i\) и исследовании условий, при которых точные и приближенные решения близки.

Замечание

Все излагаемые далее результаты справедливы для систем дифференциальных уравнений, т. е. случая, когда \(y \in R^n\), что дает возможность использовать излагаемые методы также для численного решения дифференциальных уравнений высших порядков.
