Линейные разностные уравнения и их устойчивость

Рассмотрим линейные однородные разностные уравнения порядка \(k\) с постоянными коэффициентами вида: \[y_n + a_1y_{n-1} + \cdots + a_ky_{n-k} = 0.\] Если заданы начальные условия - числа \(y_0, y_1, \cdots, y_{k-1},\) то последующие члены последовательности \(y_k, y_{k+1}, \cdots\) определяются по явной формуле \[y_n = -(a_1y_{n-1} + \cdots + a_ky_{n-k})\] и решением является последовательность \(\{y_n, n = 0, 1, \cdots\}.\)

Отметим некоторые свойства решений:

\(1.\) Среди решений есть тривиально \(\{y_n = 0, n = 0, 1, \cdots\}.\)

\(2.\) Любая линейная комбинация решений является решением.

\(3.\) Все решения представляют собой линейную комбинацию линейно независимых решений, называемых фундаментальной системой решений.

Для нахождения фундаментальной системы решений будем искать решение в виде \(y_n = \rho^n,\) получаем уравнение \[\rho^n + a_1\rho^{n-1} + \cdots + a_k\rho^{n-k} + 0,\] или, после сокращения на \(\rho^{n-k}\), алгебраическое уравнение \[\rho^k + a_1\rho^{k-1} + \cdots + a_k = 0.\] Это уравнение называется \(\textbf{характеристическим}\).

Если \(\rho\)- корень характеристического уравнения (возможно, комплексный), то ему соответствует решение - последовательность \(y = \{1,\rho,\rho^2,\cdots,\rho^n,\cdots\}\), которая остается ограниченной тогда и только тогда, когда \(|\rho| \leq 1\).

Если же корень \(\rho\) имеет кратность \(l > 1\), то ему соответствует \(l\) линейно независимых решений вида \[y^{(1)} = \{1,\rho,\rho^2,\cdots,\rho^n,\cdots\},\] \[y^{(2)} = \{0,1\rho,2\rho^2,\cdots,n\rho^n,\cdots\},\] \[\cdots\] \[y^{(l)} = \{0,1^{(l-1)}\rho,2^{(l-1)}\rho^2,\cdots,n^{(l-1)}\rho^n,\cdots\},\] которые остаются ограниченными тогда и только тогда, когда \(|\rho| < 1.\)

Тривиальное решение разностного уравнения называется \(\textbf{устойчивым}\), если для любого \(\varepsilon > 0\) найдется \(\delta > 0\) такое, что для всякого решения разностного уравнения \(y = \{y_0,y_1,\cdots,y_n,\cdots\}\) с начальными условиями \(|y_i| < \delta, i = 0,1,\cdots,k-1,\) выполняется \(|y_i| < \varepsilon, i = k, k+1, \cdots\).

В этом случае говорят также, что \(\textbf{разностное уравнение устойчиво}\).

Из представлений фундаментальной системы решений вытекает утверждение.

Критерий устойчивости

Для того чтобы разностное уравнение было устойчивым, необходимо и достаточно, чтобы для всех корней характеристического уравнения выполнялось условие \[|\rho| \leq 1,\] причем если выполняется \(|\rho| = 1\), то корень не кратный.
