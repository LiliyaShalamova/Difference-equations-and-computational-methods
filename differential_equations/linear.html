Линейные разностные уравнения

\(\textbf{Линейным разностным уравнением n-го порядка с переменными коэффициентами}\) называется уравнение \[x(t + n) + a_1(t)x(t + n - 1) + \cdots + a_n(t) x(t) = b(t),\] где \(a_1(t),\cdots ,a_n(t), b(t)\) - некоторые функции от номера \(t \in N\). Полагаем \(a_n(t) \neq 0\) для всех \(t.\) Если \(b(t) \equiv_{t \in N} 0\), то уравнение называется \(\textbf{линейным однородным}\).

Рассмотрим сначала линейное разностное однородное уравнение первого порядка \[x(t + 1 ) + a_1(t)x(t) = 0.\] Примем теперь, что \(t \in N_0 = N \cup {0}\). Для решения уравнения произведем последовательно подстановки, считая, что задан \(x(0)\). Имеем \[x(1) = -a_1(0)x(0),\] \[х(2) = -a_1(1)x(1) = a_1(0)a_1(1)x(0),\] наконец, получим \[x(t) = (-1)^ta_1(0)a_1(1)a_1(2)\cdots a_1(t - l)x(0).\] Далее, полагая, что \(x(0) = С\), и применяя обозначения \[A(0) = 1,\] \[A(t) = ( - l )^t\prod_{j=0}^{t-1}a_1(j), t \in N,\] находим формулу: \[x(t) = CA(t).\] Это формула общего решения однородного уравнения. Заметим, что формула в общем случае содержит произведения с увеличивающимся числом сомножителей.

Линейное неоднородное разностное уравнение первого порядка \[x(t + 1) + a_1(t)x(t) = b(t)\] решим методом вариации произвольной постоянной. Будем искать его решение в том же виде, что и решение соответствующего линейного однородного уравнения, но будем считать \(С\) не произвольной постоянной, а некоторой неизвестной функцией \(C(t)\), \(t \in N_0\). Итак, решение ищем в виде \[x(t) = C(t)A(t),\] где функцию \(C(t)\) находим подстановкой в уравнение.

Тогда \[C(t + 1 ) A(t + 1 ) + a_1(t)C(t)A(t) = b(t).\] Поскольку \[A(t + l) = -a_1(t)A(t),\] получаем \[C(t + 1 ) A(t + 1) — C(t) A(t + 1) = b(t),\] \[C(t + 1) = C(t) + \frac{b(t)}{A(t+1)}.\] Напомним, что в силу определения линейного неоднородного уравнения \(A(t + 1) \neq 0\) .

Снова производя последовательные подстановки, получим \[C(t) = C(0) + \sum_{j=0}^{t-1}\frac{b(j)}{A(j+1)}.\] Полагая, что \(С(0 ) = D \)(\(D\) - произвольная постоянная), и подставляя \(C(t)\), находим общее решение исходного неоднородного уравнения \[x(t) = [D + \sum_{j=0}^{t-1}\frac{b(j)}{A(j+1)}]A(t).\] Заметим, что, как и в случае линейного неоднородного обыкноного неоднородного разностного уравнения представляет собой сумму общего решения соответствующего линейного однородного уравнения и частного решения данного неоднородного уравнения.

Пример

Решим уравнение \[x(t) - (\frac{t+2}{t+1})^2x(t) = \frac{2t+4}{t+3}.\]

\(\textbf{Решение.}\) Положим \(А(0 ) = 1\) , вычислим при \(t \in N\) \[A(t) = (-1)^t\prod_{j=0}^{t-1}(\frac{j+2}{j+1})^2 = (t+2)^2,\] \[\sum_{j=0}^{t-1}\frac{b(j)}{A(j+1)} = 2\sum_{j=0}^{t-1}\frac{j+2}{(j+3)(j+2)^2} = \] \[= 2\sum_{j=0}^{t-1}\frac{1}{(j+3)(j+2)} = 2 \sum_{j=0}^{t-1}(\frac{1}{j+2} - \frac{1}{j+3}) = \] \[= 2(\frac{1}{2} - \frac{1}{t+2}) = \frac{t}{t+2}.\] Общее решение имеет вид \[x(t) = (D + \frac{t}{t+2})(t+1)^2,\] где \(D\) — произвольная постоянная. Отметим, что в рассмотренном примере удалось записать общее решение в простом виде: формула не содержит произведения с увеличивающимся числом сомножителей.
