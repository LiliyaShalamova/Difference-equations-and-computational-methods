Основные определения

Разностным (или рекуррентным) уравнением называют уравнение относительно неизвестной последовательности. Разностные уравнения часто используются в моделях динамики с дискретным временем и для приближенного решения дифференциальных уравнений.

В экономике к процессам с дискретным временем относятся ежедневные изменения курса валюты, стоимости некоторых ценных бумаг, величины банковских вкладов; ежемесячный расчет зарплаты и процентов по вкладам; ежеквартальные, ежегодные статистические данные о других микроэкономических и макроэкономических показателях.

\(\textbf{Разностным уравнением n-го порядка}\) называется уравнение вида \[x(t + n) = F (t, x(t), x(t + 1 ), \cdots, x(t + n - 1 ))(2 .1 .1),\] где \(t \in N\), \(F : N \times X^n \rightarrow X\) — заданная функция, \(X \subseteq R\) .

\(\textbf{Решением}\) разностного уравнения называется всякая функция \(\varphi: N \rightarrow X\) , для которой \[\varphi(t + n) = F(t, \varphi(t), \varphi(t + 1), \cdots, \varphi(t + n - 1))\] при всех \(t \in N\).

Замечательным свойством рекуррентного уравнения (2 .1 .1) является то, что оно само задает алгоритм построения решения. А именно, имея \(n\) начальных условий \(x(1), x(2) ,\cdots, x(n)\), по формуле (2 .1 .1) находим \[x(n + 1) = F(1 , x(1), x(2), \cdots, x(n))\] и далее по значениям \(x(2 ), x(3), \cdots, x(n + 1 )\) определяем \[x(n + 2 ) = F(2, x(2), x(3), \cdots, x(n+ 1)).\] На \(s\)-й итерации, зная предыдущие \(n\) значений \(x(s - n + 1 ) ,\cdots, x(s)\), вычисляем \[x(s + 1) = F(s - n + 1, x(s - n + 1), x(s - n + 2), \cdots, x(s)).\]

Как видно, этот алгоритм будет работать и решение будет существовать, если область определения функции \(F\) согласована с ее областью значений \(F : N \times Х^n \rightarrow X\) и начальные значения \(x(1 ), x(2 ) ,\cdots, x(n)\) принадлежат \(F\), т. е. если на каждом шаге возможна описанная подстановка. Например, для уравнения \(x(t + 1) = - \sqrt{x(t)} - 1\) уже на первом шаге получим \(x(2 ) < 0\), которое не входит в область определения функции, стоящей в правой части этого уравнения.

Замечание

Если в правую часть \(F\) уравнения (2.1.1) явно не входят аргументы \(x(t), x(t + 1 ), \cdots, x(t + р - 1 )\), \(1 \leq p \leq n-1 \), т.е. если уравнение имеет вид \[x(t + n) = F(t, x(t + р), \cdots, x(t + n - 1 )),(2.1.2) \] то заменой времени \(\tau = t + р\) оно сводится к уравнению меньшего порядка \[x(\tau + n - p) = F(t, x(\tau), \cdots, x(\tau + n - p - 1)).(2.1.3) \] При этом от \(p\) начальных значений \(x(1 ), x(2), \cdots, x(p)\) для исходного уравнения (2.1.2) решение уравнения (2.1.3) зависеть не будет. Пусть \(\varphi(\tau) (\tau \in N)\) - решение уравнения (2.1.3) такое, что \(\varphi(1 ) = x(p + 1 ), \cdots, \varphi(n - p) = x(n)\), тогда не составляет труда выписать решение уравнения (2 .1 .2 ) в виде последовательности \[x(1), x(2), \cdots, x(p), \varphi(1), \cdots, \varphi(n), \cdots (2.1.4)\] В дальнейшем будем предполагать, что уравнение (2.1.1) уже приведено к виду (2.1.3).

Как и для обыкновенных дифференциальных уравнений, для разностных уравнений ставится \(\textbf{задача Коши}\).

Теорема (о существовании и единственности решения разностной задачи Коши)

Для любого набора начальных значений \((u_1, u_2 , \cdots, u_n) \in X^n\) существует единственное решение уравнения \(n\)-го порядка (2 .1 .1 ), удовлетворяющее начальным условиям \(\varphi(1 ) = u_1, \varphi(2) = u_2,\cdots, \varphi(n) = u_n\).
