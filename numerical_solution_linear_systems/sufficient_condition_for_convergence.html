Достаточные условия метода простой итерации для решения систем линейных уравнений

Приведенные необходимые и достаточные условия сходимости метода простой итерации не совсем удобны: они связаны со сложной задачей нахождения всех собственных значений матрицы. Следующее вспомогательное утверждение дает путь получения многих простых достаточных условий сходимости.

Лемма

Модуль собственного числа матрицы не превосходит нормы матрицы:\[|\lambda(B)| \leq ||B||.\]

\(\textbf{Доказательство.}\)

Пусть \(\lambda\) – собственное число матрицы \(B\), а \(x\) – соответствующий собственный вектор, тогда по определению \(Bx = \lambda x\). Из свойств подчиненной нормы вытекает \[\left \| B \right \| \left \| x \right \| \geq ||Bx|| = |\lambda x| = |\lambda|||x||.\]

Так как собственный вектор \(x\) ненулевой, его норма не ноль, поэтому можно в неравенстве сократить на \(||x||\), получим требуемое.

Теорема

Если \(||B|| < 1\), то метод простой итерации сходится при любом начальном приближении.

\(\textbf{Доказательство}\) получается из неравенства \[|\lambda(B)| \leq ||B|| < 1.\]

В частности, метод простой итерации сходится, если выполняется одно из следующих условий:

\(1.\) Для любого \(j = 1, \cdots , n\) \[\sum_{i=1}^n|b_{ij}| < 1\]

\(2.\) \[\sum_{i,j=1}^nb_{ij}^2 < 1\]

\(3.\) Для любого \(i = 1, \cdots , n\) \[\sum_{j=1}^n|b_{ij}| < 1\]
