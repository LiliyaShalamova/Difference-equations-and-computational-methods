Метод Гаусса-Зейделя

Посмотрим на покоординатные формулы метода Якоби. Когда считается вторая координата \(k+1\)-го приближения, уже известно \(x^{(k+1)}_1\), поэтому в формулах естественно использовать уже посчитанные приближения координат. Получаем метод, называемый \(\textbf{методом Гаусса-Зейделя}\) \[ \left\{\begin{matrix} x_1^{(k+1)} = - \frac{a_{12}}{a_{11}}x_2^{(k)} - \cdots - \frac{a_{1n}}{a_{11}}x_n^{(k)} + \frac{b_1}{a_{11}}, \\ x_2^{(k+1)} = - \frac{a_{21}}{a_{22}}x_1^{(k+1)} - \cdots - \frac{a_{2n}}{a_{22}}x_n^{(k)} + \frac{b_2}{a_{22}}, \\ ...................................... \\ x_n^{(k+1)} = - \frac{a_{n1}}{a_{nn}}x_1^{(k+1)} - \frac{a_{n2}}{a_{nn}}x_2^{(k+1)} - \cdots + \frac{b_n}{a_{nn}}. \end{matrix}\right. \]

Перепишем его в матричной форме, используя те же обозначения, что и в методе Якоби: \[x^{(k+1)} = D^{−1}(−Lx^{(k+1)} − Rx^{(k)} + b).\] Чтобы свести к методу простой итерации, приведем подобные при \(x^{(k+1)}\):

\[Dx^{(k+1)} + Lx^{(k+1)} = −Rx^{(k)} + b, x^{(k+1)} =\] \[= −(L + D)^{−1}Rx^{(k)} + (L + D)^{−1}b,\] откуда матрица \(B\) в методе простой итерации определяется соотношением \[B = −(L + D)^{−1}R.\]

Теорема

Пусть \(a_{ii} \neq 0\) для всех \(i = 1, \cdots , n\). Тогда для сходимости метода Гаусса-Зейделя необходимо и достаточно, чтобы все корни уравнения \[det(\lambda L + \lambda D + R) = 0\]удовлетворяли соотношению \(|\lambda| < 1\).

\(\textbf{Доказательство.}\)

Согласно доказанному критерию сходимости метода простой итерации, метод сходится тогда и только тогда, когда все собственные числа матрицы \(B\) по модулю меньше единицы. Выпишем характеристическое уравнение, учитывая конкретный вид матрицы в методе Гаусса-Зейделя и проделаем преобразования \[0 = det(\lambda E − B) = det(\lambda E + (L + D)^{−1}R) =\] \[= det((D + L)^{−1})det(\lambda(L + D) + R),\]откуда следует заключение теоремы.

Без доказательства приведем два достаточных условия сходимости метода Гаусса-Зейделя.

Теорема

Если исходная матрица \(A\) имеет диагональное преобладание, то метод Гаусса-Зейделя сходится.

Теорема

Если исходная матрица \(A\) симметричная и положительно определена, то метод Гаусса-Зейделя сходится.
