Элементарные квадратурные формулы

Элементарные квадратурные формулы (Формулы Ньютона-Котеса) – это широко распространенные интерполяционные квадратуры при небольшом количестве узлов и при их стандартном расположении.

Их изучение начнем с \(n = 0\) (один узел).

\(\textbf{Формулы прямоугольников.}\) Рассмотрим три случая.

\(\textbf{A.}\) \(x_0 = a\). Тогда \(L_0(x) = f(x_0) = f(a)\), \[\int_{a}^{b}f(a)dx = f(a)(b − a)\]и формула имеет вид \[\int_{a}^{b}f(x)dx \approx f(a)(b − a),\] эта формула называется \(\textbf{элементарной формулой левых прямоугольников}\).

Алгебраическая степень точности \(N = 0\), как можно проверить по определению.

\(\textbf{Б.}\) \(x_0 = b\). Тогда \(L_0(x) = f(b)\), \[\int_{a}^{b}f(b)dx = f(b)(b − a)\]и формула имеет вид \[\int_{a}^{b}f(x)dx \approx f(b)(b − a),\] эта формула называется \(\textbf{элементарной формулой правых прямоугольников}\).

Алгебраическая степень точности \(N = 0\).

\(\textbf{В.}\) \(x_0 =\frac{a+b}{2}\). Тогда \(L_0(x) = f(\frac{a+b}{2})\), \[\int_{a}^{b}f(a)dx = f(\frac{a + b}{2})(b − a)\]и формула имеет вид \[\int_{a}^{b}f(x)dx \approx f(\frac{a + b}{2})(b − a),\] эта формула называется \(\textbf{элементарной формулой средних прямоугольников}\).

Алгебраическая степень точности \(N = 1\). Это можно проверить, подставляя в формулу последовательно \(1, x, x^2\). .

\(\textbf{Формула трапеций.}\) Пусть теперь \(n = 1\) (два узла), \(x_0 = a, x_1 = b\). Тогда \(L_1(x) = f(a) + f(a, b)(x − a) = f(a) + \frac{f(b)−f(a)}{b−a}(x − a)\), \[\int_{a}^{b}(f(a) + \frac{f(b) − f(a)}{b − a}(x − a))dx = f(a)(b − a) +\] \[+ \frac{f(b) − f(a)}{b − a}\frac{(b − a)^2}{2} = \frac{f(a) + f(b)}{2}(b − a),\] и формула имеет вид \[\int_{a}^{b}f(x)dx \approx \frac{f(a) + f(b)}{2}(b − a).\] Эта формула называется \(\textbf{элементарной формулой трапеций}\).

Алгебраическая степень точности \(N = 1\).

\(\textbf{Формула Симпсона.}\) Пусть \(n = 2\) (три узла), \(x_0 = a\), \(x_1 =\frac{a+b}{2}\), \(x_2 = b\). Тогда \(L_2(x) = f(a) + f(a, \frac{a+b}{2})(x − a) + f(a, \frac{a+b}{2}, b)(x − a)(x − \frac{a+b}{2})\).

Чтобы не проделывать дальнейшее громоздкое расписывание разделенных разностей и последующее интегрирование, воспользуемся методом неопределенных коэффициентов. Запишем формулу \[\int_{a}^{b}f(x)dx \approx Af(a) + Bf(\frac{a + b}{2}) + Cf(b).\]

Подставим \(f(x) \equiv 1\), получим равенство \[b − a = A + B + C\] Подставим f(x) = x − a, получим равенство \[\frac{(b − a)^2}{2} = B\frac{(b − a)}{2} + C(b − a)\] Подставим \(f(x) = (x − a)^2\), получим равенство \[\frac{(b − a)^3}{3} = B\frac{(b − a)^2}{4} + C(b − a)^2\]

Решая эту систему, получим \(A = \frac{1}{6}(b − a)\), \(B = \frac{4}{6}(b − a)\), \(C = \frac{1}{6}(b − a)\), т.е.формула приобретает вид \[\int_{a}^{b}f(x)dx \approx \frac{b − a}{6}(f(a) + 4f(\frac{a + b}{2}) + f(b)),\] эта формула называется элементарной интерполяционной формулой парабол или \(\textbf{элементарной формулой Симпсона}\).

Алгебраическая степень точности \(N = 3\). Это можно проверить, подставляя в формулу последовательно \(1, x, x^2, x^3, x^4\).

\(\textbf{Формула "3/8"}\).

Пусть \(n = 3\), четыре равноотстоящих узла, расстояние между которыми \(\frac{b−a}{3}\), тогда \(x_0 = a\), \(x_1 = \frac{2a+b}{3}\), \(x_2 = \frac{a+2b}{3}\), \(x_3 = b\).

Запишем формулу с неопределёнными коэффициентами \[\int_{a}^{b}f(x)dx \approx Af(a) + Bf(\frac{2a + b}{3}) + Cf(\frac{a + 2b}{3}) + Df(b).\] Коэффициенты определяются, подставляя многочлены низших степеней, в результате \(A = D = \frac{b−a}{8}\), \(B = C = \frac{3(b−a)}{8}\).

Полученная формула \[\int_{a}^{b}f(x)dx \approx \frac{b − a}{8}(f(a) + 3f(\frac{2a + b}{3}) + 3f(\frac{a + 2b}{3}) + f(b))\] называется \(\textbf{формулой "3/8"}\).

Заметим, что с увеличением числа узлов элементарные интерполяционные квадратурные формулы всё усложняются, что создает препятствия для их практического применения. Для повышения точности получил распространение другой подход – составные квадратурные формулы.
