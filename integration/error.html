Погрешность интерполяционных квадратурных формул

Для краткости будем применять следующие обозначения \[I[f] = \int_{a}^{b}f(x)dx, S[f] = \sum_{k=0}^{n}A_kf(x_k),\] тогда по определению погрешность интегрирования \[R_{инт}[f] = I[f] − S[f],\] а для интерполяционных квадратур, у которых \(S[f]\) получено интегрированием интерполяционного многочлена \[R_{инт}[f] = I[f] − I[L_n] = I[f − L_n] = I[R_n].\] Подставляя формулу для погрешности интерполяции получаем \[R_{инт}[f] = \int_{a}^{b}\frac{f^{(n+1)}(ξ)}{(n + 1)!}\omega_n(x)dx.\] Заметим, что \(ξ\) зависит от \(x\), поэтому оценить интеграл в общем случае затруднительно. Однако, если мы перейдем к модулю погрешности численного интегрирования, то получим оценку \[|R_{инт}[f]| \leq \frac{M_{n+1}}{(n + 1)!}\int_{a}^{b}|\omega_n(x)|dx.\]
